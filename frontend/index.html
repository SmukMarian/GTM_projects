Haier Project Tracker

Локальное веб-приложение для ведения продуктовых групп, проектов и GTM-этапов. Текущая страница — временная заглушка до подключения полноценного фронтенда.

Статус

  • Базовая структура проекта создана (backend + frontend каталоги).
  • Запуск сервера: uvicorn backend.app.main:app --reload.
  • Фронтенд обслуживается как статический каталог.

Следующие шаги

  • Спроектировать модель данных и хранилище.
  • Реализовать экраны и навигацию согласно ТЗ.
  • Добавить импорт/экспорт через Excel и резервное копирование.
